用户可以把论坛上的不良主题或优秀主题反映给管理人员,管理人员根据用户反映的记录,把优秀的帖子加为精华帖,删除不良主题.

数据库表:pw_report(tid,pid,uid,type,reason)
前台:job.php 插入"反映"记录(判断是否已经反映过) 选项:不良主题/优秀主题
后台:report.php 管理"反映"记录

七、首页调用管理

1. 使用 PHPWind首页调用 功能前需要先在 new.php 文件中设置好相关参数，文件里附有说明。
2. 首页调用方式分为 “论坛版块调用”，“论坛公告调用”，“论坛基本信息调用”，“论坛会员排行调用”，“论坛文章调用”。 
3. 每种调用方式都提供了一系列的参数设置，用户可根据需要自定义设置。 

论坛版块调用

可选参数和参数说明：
$pre : 前缀,可选参数 : 0.<li>，1.◇，2.・，3.○，4.●，5.-，6.□-
$fidin : 版块ID，设置需要调用的版块ID，格式：1_2_3

论坛公告调用

可选参数和参数说明：
$pre : 前缀,可选参数 : 0.<li>，1.◇，2.・，3.○，4.●，5.-，6.□-
$num : 调用公告个数
$length : 公告标题长度
$date : 是否显示公告发表日期，可选参数：0，1

论坛信息调用

可选参数和参数说明：
$pre : 前缀,可选参数 : 0.<li>，1.◇，2.・，3.○，4.●，5.-，6.□-
$member : 是否显示论坛会员总数和最新注册会员，可选参数：0，1
$article : 是否显示主题数和帖子数，可选参数：0，1
$yesterday : 是否显示今日，昨日，最高日发帖，可选参数：0，1
$online : 是否显示在线信息，可选参数：0，1

论坛会员调用

可选参数和参数说明：
$pre : 前缀,可选参数 : 0.<li>，1.◇，2.・，3.○，4.●，5.-，6.□-
$num : 调用会员个数
$order : 排序方式，可选参数 : 1.注册日期 2.发帖数 3.精华数 4.威望 5.财富 6.支持度

论坛文章调用 

可选参数和参数说明：
$pre : 前缀,可选参数 : 0.<li>，1.◇，2.・，3.○，4.●，5.-，6.□-
$num : 调用主题个数
$length : 文章标题长度
$order : 排序方式，可选参数 : 1.最后发表日期 2.主题发表日期 3.回复数 4.查看数
$fidin : 调用文章的版块ID，格式：1_2_3
$fidout : 不调用的版块ID，格式：1_2_3
$digest : 调用精华帖，可选参数：0，1
$postdate : 是否显示文章发表日期，可选参数：0，1
$author : 是否显示文章作者，可选参数：0，1
$replies : 是否显示回复数，可选参数：0，1
$hits : 是否显示查看数，可选参数：0，1
$fname : 是否显示版块名称，可选参数：0，1

论坛群功能介绍：

　　“论坛群”是为用户用于交流经验，讨论技术，休闲娱乐等活动而设计的功能插件。

注册新群：

　　每个用户可以注册一个或多个群（后台控制），群注册成功后即可享受群个各种功能。

我要捐献：为群捐献金钱，增加群的影响力
我要退出该群：退出该群

群管理功能：

群基本设置：修改群的基本信息 
群成员管理：添加或删除群会员
管理员设置：添加或删除群管理员
解散该群：解散该群
